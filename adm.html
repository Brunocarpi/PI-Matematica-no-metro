Instruções para Instalação

Instalando o Visual Studio Code (VS Code)

Siga os passos abaixo para configurar o ambiente de desenvolvimento:

  1. Baixe e instale o VS Code: https://code.visualstudio.com/
  2. Após a instalação, abra o terminal integrado do VS Code.
  3. Instale o Node.js (caso não tenha instalado):
    4. sudo apt install nodejs
  4. Instale o NPM (gerenciador de pacotes):
    5. sudo apt install npm
  5. Clone o repositório do projeto:
    6. git clone https://github.com/Brunocarpi/PI-Matematica-no-metro.git
  6. Acesse a pasta do projeto:
    7. cd PI-Matematica-no-metro
  7. Instale as dependências do projeto:
    8. npm install
  8. Para rodar o servidor, use o comando:
    9. npm start
  9. Abra o navegador e acesse o site em:
    10. http://localhost:5000

Pronto para utilizar!

Com esses passos, você terá o projeto configurado localmente.
